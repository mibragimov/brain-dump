# Brain Dump

Just type. Don't think. Don't organize.

## What it does

- Large text area to dump everything in your head
- Auto-saves as you type
- Save dumps with timestamps
- Load previous dumps
- Copy to clipboard
- Everything stored in localStorage

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Lucide React

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm start
```
